using to-markdown and corrected in Atom using these regular expressions:

```
<[^\>]*>  => (empty)
https://www\.google\.com/url\?q=([^&]*)[^\)]*  => $1
```

Convertable to CSV using this regexp:
```
\[([^\]]*)\]\(([^\)]*)\)\W+\(([^,]*)([^\)]*)\W*(.*)  => $1;$2;$3;$4;$5
^[A-Z]$  => (empty)
;,\W  => (empty)
```

Sort, crop & save. Salt, pepper & enjoy.
